# GPA Calculator

A simple web-based **GPA Calculator** that helps students quickly estimate their GPA by entering their subject grades. The tool calculates both **raw GPA and rounded GPA** and provides a **letter-grade and percentage reference chart**.

---

## Features

- Calculate GPA from multiple subjects
- Displays **Raw GPA** and **Rounded GPA**
- Includes a **GPA → Letter Grade → Percentage** reference table
- Quick **Grade to Marks estimator**
- Simple and lightweight (runs directly in the browser)
- No data collection or storage

## How It Works

1. Open the calculator in your browser.
2. Click **Add Subject** to add classes.
3. Enter your grades.
4. The calculator automatically updates and shows:
   - Raw GPA
   - Rounded GPA
   - Grade equivalents
